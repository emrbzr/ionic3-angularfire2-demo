# Ionic3 with angularfire2 (For FireBase)



### Data to be changed with your own info,
In `src/app/app.module.ts`
```ts
export const firebaseConfig = {
  apiKey: "",
  authDomain: "",
  databaseURL: "",
  projectId: "",
  storageBucket: "",
  messagingSenderId: ""
};
```

### To install
(Make sure you have latest NodeJS)
```
npm install
```
